# RWFramework

A framework that can be used in conjunction with KMFramework to add additional options for objects in KTaNE Rewritten.

To use, include RWFramework.dll (and optionally RWFramework.xml) in the Assets/Plugins/Managed folder of your modkit project.

- [RWFramework.dll](https://github.com/KtaneRewritten/RWFramework/releases/latest/download/RWFramework.dll)
- [RWFramework.xml](https://github.com/KtaneRewritten/RWFramework/releases/latest/download/RWFramework.xml)
